# Data Folder

This folder contains sample datasets used for testing and demonstration purposes.

⚠️ Real evaluation datasets are not included in this repository due to:

- Privacy protection
- Ethical considerations
- Institutional confidentiality

If you want to test the pipeline, you can create a CSV file with the following structure:

Column example:

| feedback_text | sentiment_label |
|---------------|-----------------|
| Great teacher | Positive        |
| Confusing     | Negative        |

Supported classification modes:

- Binary (Positive / Negative)
- Multiclass (Negative / Neutral / Positive)

The system is designed to be adaptable to educational and organizational survey analytics.
